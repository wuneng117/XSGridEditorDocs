# 在场景中编辑网格

场景编辑里主要是通过扩展2D Tilemap Editor的GameObjectBursh为GameObjectBrushEx，让画的prefab会自动贴到装饰物的上方（毕竟网格肯定在最上面嘛），使用prefab来画网格，用prefab是因为一般网格是用贴花来做的，这样网格可以贴合到地表，怎么画可以参考下别人的这个视频
